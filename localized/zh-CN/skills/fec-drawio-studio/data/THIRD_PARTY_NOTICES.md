此 skill 包含用于生成可编辑图表的精简索引和 manifest。

- `shape-index.json` 包含用于本地样式检索的 draw.io 图形元数据。图形名称、尺寸和 style 字符串来自 draw.io / diagrams.net 图形生态，并继续受其上游条款约束。
- `brand-icons.json` 是 lobe-icons CDN 资产的 manifest。lobe-icons 使用 MIT 许可；各个 logo 和名称可能是其各自所有者的商标。
- `brand-symbols.mjs` 可能为部分基础设施和数据库符号引用 simple-icons CDN slug。simple-icons 元数据和图标遵循其上游许可；品牌商标仍归各自权利人所有。
- 工作流文档提到可选地与外部 draw.io MCP App/Tool 部署互操作。这些服务未被本包打包、内置或设为必需。

这些资源仅用于识别和渲染图表符号。重新分发包时，请保留此声明。
