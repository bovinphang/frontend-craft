# frontend-craft: qoder

See root README **Universal Install** and run:

```bash
npx @bovinphang/frontend-craft install qoder
```

> **Non-interactive:** append `--local` or `--global` to skip install-location prompts. Local installs write to `.qoder/`; global installs write to `~/.qoder` or `QODER_CONFIG_DIR`.

Installed assets:

- `skills/` -> `.qoder/skills/`
- `commands/` -> `.qoder/commands/`
- `agents/` -> `.qoder/agents/`
- shared rules -> `.qoder/rules/`
- hook scripts -> `.qoder/hooks/`
- hook configuration -> `.qoder/settings.json`
