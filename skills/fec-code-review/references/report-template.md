# 代码审查报告模板

```markdown
# 代码审查报告

> 生成时间: YYYY-MM-DD HH:mm
> 评审工具: frontend-craft

## CRITICAL / 必须修改 (N项)
- **[文件:行号]** 问题描述 -> 建议修改

## HIGH / 建议优化 (N项)
- ...

## MEDIUM / 可选优化项 (N项)
- ...

## LOW / 观察项 (N项)
- ...

## 做得好的地方
- ...

## 风险等级：低 / 中 / 高

**总体评价**: 可合并 / 待修改后合并 / 需要重构
```

评审完成后，将报告保存到 `reports/code-review-YYYY-MM-DD-HHmmss.md`，并告知用户报告路径。
