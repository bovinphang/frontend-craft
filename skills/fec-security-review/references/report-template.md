# 安全审查报告模板

```markdown
# 安全审查报告

> 生成时间: YYYY-MM-DD HH:mm
> 评审工具: frontend-craft

## CRITICAL / 高危 (N项)
- **[文件:行号]** 风险描述 -> 修复建议

## HIGH / 中高危 (N项)
- ...

## MEDIUM / 中危 (N项)
- ...

## LOW / 低危或建议 (N项)
- ...

## 已通过的安全检查
- ...

**整体安全等级**: 安全 / 存在风险 / 高危需修复
```

审查完成后，将报告保存到 `reports/security-review-YYYY-MM-DD-HHmmss.md`，并告知用户报告路径。
